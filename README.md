# 🎓 Student Management API

A secure REST API built using **FastAPI**, **SQLite**, **SQLAlchemy**, and **JWT Authentication**.

This project demonstrates professional backend development concepts including CRUD operations, secure authentication, modular routing, API validation, password hashing, and environment variable configuration.

---

# 🚀 Features

- User Registration
- User Login
- JWT Authentication
- Protected APIs
- Student CRUD Operations
- SQLite Database
- SQLAlchemy ORM
- Password Hashing
- Request Validation
- Environment Variables
- Modular Project Structure
